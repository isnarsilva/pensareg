# PensaReg — página institucional

Modernização da página inicial da PensaReg em HTML5, CSS3 e JavaScript ES6+, sem frameworks ou dependências externas.

## Uso

Abra `index.html` em um navegador moderno ou sirva esta pasta por HTTP estático.
